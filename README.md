# Music-Management

<hr>

## Installation 

### Create an environment
```
python -m venv env
```

```
source env/bin/activate
```

### Clone Repo
```
https://github.com/Apex1000/Music-Management.git
```
```
cd Music-Management
```

### Install dependencies
```
pip install -r requirements.txt
```
### Run application
```
flask run
```
